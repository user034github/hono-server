React oppsett

1) Installert Intro React og Intro Next og kopiert innholdet i <body> 
   fra index.html til return statement i App.tsx

2) Kopiert css koden fra index.html til egen css-fil i samme directory som
   App.tsx og importert css-filen i App.tsx

3) Slettet index.html fra Hono server folderen mens JSON filen og server.js
   forblir siden de er en del av backend

4) Endret API rute "/prosjekter" til "http://localhost:3000/prosjekter" i
   App.tsx for å spesifiere porten til Hono serveren og endret path til
   index.html i server.js siden den nå er i React folderen

5) Lagt til kode i server.js for å la forespørsler fra React gå gjennom i
   stedet for å bli stoppet av CORS policy

Fra de tingene som er nevnt i oppgavene, så har jeg valgt å inkludere:

1) Komponenter
2) Props
3) Fragmenter
4) Logikk
5) Events
6) State   

Og derfor har jeg gjort følgende:

1) Lagt til Component for tekst som vises i prosjekt oversikten.

2) Flyttet prosjektformen til eget Component, og endret App.tsx slik at ProsjektformComponent tar imot
   handleSubmit funksjonen som en prop. Gjorde deretter endring i ProsjektformComponent slik at
   den kan ta imot props og kalle handleSubmit.

3) Lagt til <React.Fragment> rundt return statement siden det er en relasjon mellom
   elementene i return statement og da går de i samme fragment.

4) Lagt til logikk i OversiktComponent slik at teksten på nettstedet endrer seg basert på om
   det finnes prosjekter eller ikke.

5) Lagt til en knapp som kan trykkes og som endrer tekst basert på tilstanden eller state,
   og her brukes det onClick for å registrere hendelsen. Når state endres, så blir komponenten
   re-rendered og da er det mulig å se endringen i user interface.

6) Viser antall prosjekter som vises på nettstedet ved bruk av {prosjekter.length}

7) Oppdatert server.js slik at det legges til nytt id for hvert prosjekt og lagt til
   app.delete for å kunne slette prosjekter basert på id

8) Lagt til slettProsjekt funksjon i App.tsx som er koblet til endpoint for sletting
   og som kalles ved å trykke på en knapp som gir funksjonen id'en som skal slettes
